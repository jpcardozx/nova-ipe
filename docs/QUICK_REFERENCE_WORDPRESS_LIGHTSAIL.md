## üîå Conex√£o SSH

### M√©todo 1: Via AWS CLI (chave tempor√°ria)
```bash

  --region us-east-1 \
  --instance-name Ipe-1 \
  --protocol ssh > access.json

### M√©todo 2: Usando sua pr√≥pria chave

  "mkdir -p ~/.ssh && cat >> ~/.ssh/authorized_keys"

```

### Configurar SSH config para acesso f√°cil
